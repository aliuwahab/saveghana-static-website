# Image Requirements for SAVE-Ghana Website

This folder should contain all images used throughout the website. Below is a complete list of required images organized by page.

## Recommended Image Specifications

- **Format**: JPG for photos, PNG for logos/graphics
- **Quality**: High resolution (at least 1920px wide for hero images)
- **Optimization**: Compress images before uploading (use TinyPNG or similar)
- **File naming**: Use lowercase, hyphens for spaces, descriptive names

## Image Content Suggestions

### Photography Tips
1. **People-focused**: Show community members, beneficiaries, and staff
2. **Action shots**: Capture programs and activities in progress
3. **Diverse representation**: Include various age groups, genders, and communities
4. **Authentic moments**: Candid photos often work better than posed shots
5. **Quality lighting**: Natural light works best for outdoor community shots

### Specific Image Ideas

**Hero/Header Images**:
- Community gatherings
- Landscape views of Northern Ghana
- Group activities or workshops
- Aerial views of villages

**Project Images**:
- Classrooms with students
- Healthcare workers with patients
- Women in training sessions
- Agricultural activities
- Water infrastructure
- Youth programs

**Blog Images**:
- Specific to the story being told
- Before/after comparisons
- Infographics and charts
- Maps showing project locations
- Portrait shots of community members (with permission)

## Important Notes

1. **Copyright**: Only use images you have rights to use
2. **Permissions**: Get written consent for photos of people
3. **Compression**: Always compress images before uploading
4. **Backup**: Keep original high-resolution versions
5. **Alt Text**: Remember to add descriptive alt text in HTML
6. **Diversity**: Ensure images represent all communities you serve

## File Size Guidelines

- Hero images: < 500KB (after compression)
- Card images: < 200KB
- Thumbnail images: < 150KB
- In-content images: < 300KB

## Adding Images to the Website

1. Place all images in this `/images` folder
2. Reference them in HTML using: `src="images/filename.jpg"`
3. Always include alt text: `alt="Descriptive text here"`
4. Test on different devices to ensure images load properly

## Need Help?

- Image editing: GIMP (free), Photoshop, Canva
- Compression: TinyPNG.com, Squoosh.app
- Bulk renaming: Bulk Rename Utility (Windows), Automator (Mac)
